# Обзор проекта Bonus Games

`Bonus Games` — это модуль комнатных игр на бонусных баллах.

## Что делает система

- Регистрация и логин пользователя.
- Хранение профиля и роли (`USER`, `EXPERT`, `ADMIN`).
- Учёт баланса бонусных баллов.
- Резервирование, списание и возврат бонусов.
- Создание игровых комнат по шаблонам.
- Выбор победителя через внешний `random-service`.
- Хранение истории игр и событий.
- Realtime-обновления состояния комнаты через WebSocket.

## Сервисы

- `game-backend` (Spring Boot, порт `8081`) — бизнес-логика.
- `random-service` (Spring Boot, порт `9095`) — генерация случайных чисел с replay.
- `front` (Next.js, порт `3000`) — пользовательский интерфейс.
- PostgreSQL (порт `5432`) — данные и аудит.
